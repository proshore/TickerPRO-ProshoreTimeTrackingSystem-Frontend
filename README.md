# Ticker

> Track your valuable time.

## Project setup

```sh
> git clone https://github.com/rabirajkhadka/TickerPRO-ProshoreTimeTrackingSystem-Frontend.git
> cd TickerPRO-ProshoreTimeTrackingSystem-Frontend
> npm install
> npm run dev
```
